JavaScript의 IS-A 관계

두 요소의 __proto__ 사이에 연관 관계가 형성되면 IS-A 관계
Array 타입 a의 __proto__는 Array이므로 Array와 a는 IS-A 관계(instanceof 결과 true)
Array의 __proto__는 Object이므로 Object와 a는 IS-A 관계(instanceof 결과 true)
✰__proto__ 형태의 요소를 내부 요소(Internal Property)라 칭함
원래 형태는 [[prototype]]
__proto__는 Prototype Link, Object.prototype은 Prototype Property
Link와 Property 사이에 연쇄 관계가 형성 -> prototype chain 관계
