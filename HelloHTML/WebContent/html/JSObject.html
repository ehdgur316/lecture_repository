Lexical Environment

Creation Phase의 일부(ECMA Script 3 이후)
어휘적으로 표현된 스크립트 구조와 이에 따른 Property/Prototype 관계를 의미
Environment Record와 Outer Reference로 구성

Environment Record

함수에서 선언된 모든 지역변수 정보 저장
지역변수 중 변수는 Undefined로 생성 및 초기화,
지역변수 중 함수는 함수 생성 후 함수 포인터 바인딩
(표현식 함수는 함수 포인터가 Undefined로 생성/초기화됨)

Outer Reference

함수 외부(상위 Scope)를 참조하는 것
